# 手动遥控建图指南

用键盘手动驾驶小车，在 Gazebo 中完成 SLAM 建图，并保存地图。

## 前提

- Ubuntu 18.04 + ROS Melodic
- 已编译 `scene-recognition-robot` 工作空间

## 安装依赖

```bash
sudo apt install -y ros-melodic-gmapping ros-melodic-turtlebot3-teleop
```

## 步骤一：启动建图环境

**终端 1：**

```bash
source /opt/ros/melodic/setup.bash
source ~/scene-recognition-robot/devel/setup.bash
export TURTLEBOT3_MODEL=waffle_pi
roslaunch scene_recognition_robot slam_manual.launch
```

等待 Gazebo 和 RViz 出现。

## 步骤二：键盘遥控小车

**终端 2：**

```bash
source /opt/ros/melodic/setup.bash
source ~/scene-recognition-robot/devel/setup.bash
export TURTLEBOT3_MODEL=waffle_pi
roslaunch turtlebot3_teleop turtlebot3_teleop_key.launch
```

### 按键说明

| 按键 | 功能 |
|------|------|
| `w` | 前进 |
| `s` | 后退 |
| `a` | 左转 |
| `d` | 右转 |
| `x` | 停止 |
| `空格` | 紧急停止 |

> 点击 **终端 2 窗口** 确保焦点在终端上，再按键。

## 步骤三：建图技巧

1. **慢速移动**：按住 `w` 短按，不要一直长按
2. **走遍所有房间**：卧室、餐厅、厨房、客厅、走廊都要走到
3. **沿墙行驶**：激光雷达能扫到墙壁，地图更清晰
4. **观察 RViz**：左侧勾选 **Map**，看灰色区域是否逐渐变白/变黑
5. **避免太快转弯**：转弯时用 `a`/`d` 短按

建议路线：

```
起点 → 沿墙走一圈 → 进入各房间中心 → 回到起点
```

## 步骤四：保存地图

建图满意后，**终端 3：**

```bash
source /opt/ros/melodic/setup.bash
cd ~/scene-recognition-robot
rosrun map_server map_saver -f src/scene_recognition_robot/maps/my_house_map
```

生成：

```
src/scene_recognition_robot/maps/my_house_map.pgm
src/scene_recognition_robot/maps/my_house_map.yaml
```

## 步骤五：使用自建地图导航

修改 launch 或启动时指定地图：

```bash
# 编辑 gazebo_sim.launch 中 map_server 的 args，改为：
# args="$(find scene_recognition_robot)/maps/my_house_map.yaml"

roslaunch scene_recognition_robot gazebo_sim.launch
```

## 常见问题

| 问题 | 解决 |
|------|------|
| 按键无反应 | 点击 teleop 终端窗口获取焦点 |
| RViz 无地图 | 勾选 Map；确认 gmapping 在运行 |
| 地图很乱 | 放慢速度，多走几遍 |
| Gazebo 暂停 | 点 ▶ 播放按钮 |

## 不用项目 launch 的最简方式

若 `slam_manual.launch` 不可用，分步启动：

```bash
# 终端1：仅 Gazebo（run_mission:=false，需自行去掉 map_server）
# 终端2：gmapping
rosrun gmapping slam_gmapping

# 终端3：遥控
roslaunch turtlebot3_teleop turtlebot3_teleop_key.launch

# 终端4：保存
rosrun map_server map_saver -f ~/my_map
```
